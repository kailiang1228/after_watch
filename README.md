第一次嘗試寫寫看網頁
===

>可以透過選單選擇電影、劇還是動畫 並加上自己想看的項目  

  >==單擊影片能螢光註記==
>==點擊後面的刪除按鍵則可以刪除該項目==  

  >在選擇新增項目旁有個重製鍵
>能讓項目回歸最遠始狀態
>也就是我本人想看的影片  

  >照我寫的似乎是==能在本地儲存==
>也就是你加了重新整理後還是在

>你們上傳的內容我也不會看到
